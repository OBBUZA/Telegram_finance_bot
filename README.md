# Telegram_finance_bot

Telegram-бот для анализа личных финансов на основе CSV-данных.
Поддерживает демонстрационные и пользовательские данные, строит графики и формирует отчёты.

---

## Возможности

* Анализ доходов и расходов
* Топ категорий и компаний
* Средняя трата
* Финансовые графики (линейные, круговые, гистограммы)
* Полный текстовый отчёт
* Персональные рекомендации
* Поддержка CSV-файлов
* SQLite в качестве хранилища
* Визуальная обратная связь через http.cat

---

## Запуск

```bash
pip install ...
python bot.py
```
